setlocale

来自cppreference.com
< cpp‎ | locale

语法： 

    #include <clocale>
    char *setlocale( int category, const char * locale );

setlocale函数用来设置和获取当前的locale。如果参数 locale为空指针，当前的locale将被返回。否则，用 locale 给给定的分类(category)设置locale。

参数 category 可以是以下值：

值 描述
LC_ALL locale的所有项目
LC_TIME 日期和时间格式
LC_NUMERIC 数字格式
LC_COLLATE 字符串排序和正则表达式匹配
LC_CT 正则表达式匹配，转换，大小写敏感比较，宽字符函数，和字符分类
LC_MESSAGES 自然语言信息
