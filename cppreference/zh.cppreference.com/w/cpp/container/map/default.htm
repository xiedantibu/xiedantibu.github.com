[edit]
 
在头文件 <map> 中定义
template<

    class Key,
    class T,
    class Compare = std::less<Key>,
    class Allocator = std::allocator<std::pair<const Key, T> >

> class map;
std::map是一个有序关联容器，具有独特的键包含键 - 值对。键进行排序的比较函数Compare。具有对数搜索，去除和插入操作的复杂性。地图通常实现为红黑树.
原文：
std::map is a sorted associative container that contains key-value pairs with unique keys. Keys are sorted by using the comparison function Compare. Search, removal, and insertion operations have logarithmic complexity. Maps are usually implemented as 红黑树.
