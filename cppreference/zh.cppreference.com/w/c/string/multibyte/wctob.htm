在头文件 <wchar.h> 中定义
int wctob( wint_t c );
缩小了相当于在初始位移状态，如果它的多字节字符的宽字符c是一个单字节.
原文：
Narrows a wide character c if its multibyte character equivalent in the initial shift state is a single byte.
