[edit]
 
在头文件 <threads.h> 中定义
int tss_set( tss_t tss_id, void *val );
 （C11 起）
设定值的特定于线程的存储识别tss_id为当前线程val.
原文：
Sets the value of the thread-specific storage identified by tss_id for the current thread to val.
