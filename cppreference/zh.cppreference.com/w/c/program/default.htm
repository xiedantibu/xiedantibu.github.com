[编辑] 终止程序

以下函数管理程序的终止和资源的清理。

在头文件 <stdlib.h> 中定义
abort
 引发非正常的程序终止（不清理）
（函数） [edit]
exit
 引发正常的程序终止并清理
（函数） [edit]
quick_exit
（C99）
 引发正常的程序终止但不完全清理
（函数） [edit]
_Exit
（C99）
 引发正常的程序终止但不清理
（函数） [edit]
atexit
 注册一个函数使其在调用exit()时被调用
（函数） [edit]
at_quick_exit
（C99）
 注册一个函数使其在调用quick_exit时被调用
（函数） [edit]
EXIT_SUCCESS
EXIT_FAILURE
 表示程序的执行结果
（常量宏） [edit]

[编辑] 信号

几个为信号管理提供的函数和常量宏。

在头文件 <signal.h> 中定义
signal
 为特定的信号设置信号处理函数
（函数） [edit]
raise
 运行特定信号的信号处理函数
（函数） [edit]
sig_atomic_t
 用于从异步信号处理函数访问原子实体的整数类型
（typedef）
SIG_DFL
SIG_IGN
 定义信号处理策略
（常量宏） [edit]
SIG_ERR

（常量宏） [edit]
