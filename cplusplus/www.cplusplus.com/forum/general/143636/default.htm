I think you're talking about Include Guards.
They prevent the declaring classes, functions, or variables twice.

for a "my_header.h" file:
1
2
3
4
 
#ifndef HEADER_H
#define HEADER_H
/** relevant stuff goes here, like a class definition **/
#endif
