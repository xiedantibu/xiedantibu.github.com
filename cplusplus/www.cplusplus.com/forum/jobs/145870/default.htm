C++ or C# Developer with GINA experience

anin (1)
Description

The end client is seeking to build a password reset capability into the Windows 8 login screen.

1. The goal of this project is to reduce password reset volume to the service desk.
2. The approach is to augment the capabilities of the existing self-service password reset solution Password On Demand to do the following:
3. Enable self-service password resets on the logon screen of Windows8 devices
4. Enable use of SMS-capable phones to authenticate locked out users Enabling self-service resets on the logon screen allows users to unlock and reset their passwords when and where they get locked out of their accounts. Today the users have to visit a website on their phone or visit a kiosk to perform the reset.
5. Enabling the use of SMS-capable phones for authentication lowers the barrier to registration because many people have SMS-capable phones. This is a common approach used today by cloud consumer applications (e.g. gmail) as an alternative authentication mechanism for lockouts.

Experience – 5+ Years as C++ or C# Developer

Skills Needed

Win8 Login Screen Development C++ / C# / Native Windows UI Web Service Integration AD / CA interface Skills, MFC – Microsoft Foundation Class, Authentication, Winforms, GINA, DLL – Microsoft Dynamic Link Libraries

Anyone with experience programming to the Microsoft GINA will be interviewed first
