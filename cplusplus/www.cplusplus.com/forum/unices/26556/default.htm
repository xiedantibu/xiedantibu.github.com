Starting a script with C

W2K (1)
Hi,

I am writing a small C program which should execute another script "myscript". The script "myscript" expects two parameters e.g. "panel1" and "panel2".

> myscript panel1 panel2

Afterwards, myscript needs another input which could "1.4" before the computation starts. I would like to know how I can run this out of my C-program.

chris (82)
I am not 100% about the adition input such as the "1.4" as you said, however for starting another script with parameters I suggest you look at: http://www.cplusplus.com/reference/clibrary/cstdlib/system/

This is a very simple function and can be used just like this:

system("myscript panel1 panel2");
