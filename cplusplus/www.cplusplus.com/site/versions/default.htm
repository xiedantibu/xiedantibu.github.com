std::C++ Versions

Some parts of this website have contents that only apply to specific versions of the C++ language standard.

These are preceded by one of these labels:

versiondescription
C++98C++ 1998/2003 Standard
C++11C++ 2011 Standard
Non-standardNon standard feature.

Please, check the documentation of your compiler to find out which standard it conforms to. Most compilers today comply with the C++03 standard, although support for C++11 features is growing.
